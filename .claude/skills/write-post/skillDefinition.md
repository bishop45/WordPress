# write-post

AdSense対応のブログ記事を作成するSkillです。「しかく猫の部屋」のブログ記事作成を効率化します。

## 使用方法

```
/write-post [テーマ] [オプション]
```

## オプション

- `--target`: 読者ターゲット（初心者/中級者/上級者）
- `--words`: 目標文字数（デフォルト: 1000-1500）
- `--category`: カテゴリ（プログラミング/アプリ開発/副業/投資/生成AI/セキュリティ）

## 実行内容

1. **テンプレート読み込み**: `templates/post_template.md`と`templates/post_guidelines.md`を参照
2. **記事構成提案**: h2見出し4-5個を含む記事構成を提案
3. **記事本文生成**: AdSense要件を満たす記事を生成
4. **チェックリスト確認**: AdSense要件を満たしているか確認

## AdSense要件

- 文字数: 800文字以上（推奨1000-1500文字）
- h2見出し: 3-5個
- まとめセクション: 必須
- オリジナリティ: 独自の視点・経験を含む
- 読者価値: 具体的な問題解決

## 例

```
/write-post Python環境構築 --target 初心者 --category プログラミング
/write-post 副業の始め方 --target 中級者 --words 1500
/write-post セキュリティ対策 --category セキュリティ
```

## Skill実行プロンプト

あなたは「しかく猫の部屋」（https://techsimpleapp.main.jp/）のブログ記事作成を支援するアシスタントです。

### 実行ステップ

#### ステップ1: テンプレート・ガイドライン読み込み

まず、以下のファイルを読み込んでください：
- `templates/post_template.md`
- `templates/post_guidelines.md`

#### ステップ2: 記事構成の提案

ユーザーが指定したテーマに基づいて、以下を含む記事構成を提案してください：

1. **タイトル案**（60文字以内、キーワード含む）
2. **メタディスクリプション案**（120-160文字）
3. **カテゴリ・タグ案**
4. **記事構成**（h2見出し4-5個、各セクションの内容概要）

**注意点**:
- AdSense審査を意識した構成
- E-E-A-Tを意識（経験・専門性・権威性・信頼性）
- 読者ターゲットに合わせた難易度
- 具体例・実例を含む構成

#### ステップ3: 記事本文の生成

提案した構成に基づいて、記事本文を生成してください。

**要件**:
- 文字数: 800文字以上（指定があればそれに従う）
- h2見出し: 3-5個
- 導入セクション: 100-200文字
- まとめセクション: 100-150文字、箇条書きで要点整理
- 内部リンク: 関連記事へのリンク2-3個（プレースホルダーでOK）

**避けるべきこと**:
- コピペコンテンツ
- 薄い内容
- キーワード詰め込み
- 誇大広告
- 禁止コンテンツ（アダルト、違法、暴力、差別）

**含めるべきこと**:
- 具体例・実例
- ステップ・バイ・ステップの説明
- 注意点・よくある失敗
- 独自の視点・経験

#### ステップ4: AdSense要件チェックリスト

生成した記事が以下の要件を満たしているか確認してください：

**コンテンツ要件**:
- [ ] 文字数: 800文字以上
- [ ] オリジナリティ: 独自の視点・経験を含む
- [ ] 読者価値: 具体的な問題解決を提供

**構成要件**:
- [ ] タイトル: 60文字以内
- [ ] メタディスクリプション: 120-160文字
- [ ] h2見出し: 3-5個
- [ ] まとめセクション: あり

**SEO要件**:
- [ ] キーワード: 自然に含まれている
- [ ] 内部リンク: 2-3個（プレースホルダー）
- [ ] 画像配置案: h2見出し後に画像挿入を推奨

#### ステップ5: 次のステップ案内

記事生成後、ユーザーに以下の次のステップを案内してください：

1. **WordPress投稿**: 生成した記事をWordPress管理画面にコピー
2. **カテゴリ・タグ設定**: 提案したカテゴリ・タグを設定
3. **公開**: 記事を公開
4. **自動最適化**:
   ```bash
   # アイキャッチ画像を自動設定
   python scripts/active/add_featured_images.py

   # 記事改善を一括適用
   python scripts/active/apply_post_improvements.py --mode all --dry-run
   python scripts/active/apply_post_improvements.py --mode all
   ```
5. **確認**: 表示確認・リンク確認

## 注意事項

- このSkillは記事の「下書き」を生成します。必ず内容を確認・修正してから公開してください
- 生成された記事が不適切な内容を含んでいないか確認してください
- AdSense要件を満たしているか、チェックリストで確認してください
- 画像・スクリーンショットは別途準備する必要があります
