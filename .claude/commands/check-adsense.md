# AdSense要件チェック

指定された記事がGoogle AdSense審査要件を満たしているか確認してください。

## チェック項目

以下の項目を確認し、レポートを作成してください：

### コンテンツ要件
- [ ] **文字数**: 800文字以上あるか
- [ ] **オリジナリティ**: 独自の視点・経験が含まれているか
- [ ] **読者価値**: 具体的な問題解決を提供しているか

### 構成要件
- [ ] **タイトル**: 60文字以内か
- [ ] **メタディスクリプション**: 設定されているか（120-160文字）
- [ ] **見出し構造**: h2見出しが3-5個あるか
- [ ] **まとめセクション**: 記事末尾にまとめがあるか

### 画像要件
- [ ] **アイキャッチ画像**: 設定されているか
- [ ] **本文画像**: 3枚以上あるか
- [ ] **画像altテキスト**: 設定されているか

### SEO要件
- [ ] **カテゴリ**: 設定されているか
- [ ] **タグ**: 3-5個設定されているか
- [ ] **内部リンク**: 2個以上あるか

### 禁止コンテンツチェック
- [ ] アダルトコンテンツがないか
- [ ] ギャンブル関連でないか
- [ ] 違法・暴力・差別的内容がないか

## 実行方法

1. 記事のURLまたはIDから記事を取得
2. 上記の各項目をチェック
3. 結果をマークダウン形式でレポート

## 注意事項

- このチェックは簡易的なものです。最終的にはGoogle AdSenseの審査基準に従ってください
- 記事の内容の質や独自性は、人間の目で確認することを推奨します
