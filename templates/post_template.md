# [記事タイトル] - AdSense対応記事テンプレート

> **このテンプレートについて**: Google AdSense審査に合格したブログ「しかく猫の部屋」で使用している記事構成です。このテンプレートに沿って記事を作成することで、AdSense要件を満たした記事を効率的に作成できます。

---

## 基本情報

- **タイトル**: [60文字以内、キーワードを含む]
- **カテゴリ**: [プログラミング/アプリ開発/副業/投資/生成AI/セキュリティ など]
- **タグ**: [関連タグ3-5個]
- **メタディスクリプション**: [120-160文字、記事の要約]

## 記事構成（推奨）

### 導入（100-200文字）

読者の課題・疑問を提示し、この記事で得られる価値を明確にします。

**テンプレート例**:
```
[問題提起]
「〜で困っていませんか？」
「〜を実現したいけど、どうすればいいかわからない...」

[この記事で得られること]
この記事では、〜について解説します。
〜の方法を、初心者にもわかりやすく説明していきます。
```

---

### メインコンテンツ（h2見出し × 3-5個）

各セクションは200-400文字程度。h2見出しの直後に画像を挿入することを推奨。

#### h2見出し1: [トピック1]

[内容]
- 具体的な手順
- 実例・コード例
- スクリーンショット

**💡 ポイント**: h2見出しの直後に関連画像を挿入すると、視覚的にわかりやすくなります

---

#### h2見出し2: [トピック2]

[内容]
- 詳細な説明
- 図解・表
- 注意点

---

#### h2見出し3: [トピック3]

[内容]
- 応用例
- よくある失敗
- 解決策

---

### まとめ（100-150文字）

記事の要点を3-5個の箇条書きでまとめます。

**テンプレート例**:
```
## まとめ

この記事では、[テーマ]について解説しました。

要点をまとめると以下のとおりです：

- ポイント1
- ポイント2
- ポイント3

[行動喚起や次のステップ]
```

---

### 関連記事（内部リンク）

関連する過去記事へのリンクを2-3個挿入します。

**テンプレート例**:
```
## 関連記事

- [関連記事1のタイトル](URL)
- [関連記事2のタイトル](URL)
- [関連記事3のタイトル](URL)
```

## AdSense要件チェックリスト

### コンテンツ要件

- [ ] **文字数**: 800文字以上（推奨: 1000-2000文字）
- [ ] **オリジナリティ**: コピペなし、独自の視点・経験を含む
- [ ] **読者価値**: 具体的な問題解決・学びを提供

### 画像要件

- [ ] **アイキャッチ画像**: 1枚（16:9推奨、最低1200×630px）
- [ ] **本文画像**: 3枚以上（h2見出し後に配置推奨）
- [ ] **画像クレジット**: Unsplash使用時は自動設定

### SEO要件

- [ ] **メタディスクリプション**: 120-160文字
- [ ] **タイトルタグ**: 60文字以内、キーワード含む
- [ ] **h2見出し**: 3-5個（階層構造を意識）
- [ ] **まとめセクション**: 記事の最後に必須

### 内部リンク

- [ ] **関連記事リンク**: 2-3個以上
- [ ] **自然な文脈**: 無理なリンク挿入は避ける

### カテゴリ・タグ

- [ ] **カテゴリ**: 1個（メインテーマ）
- [ ] **タグ**: 3-5個（関連キーワード）

## 自動化ツールの活用

記事公開後、以下のスクリプトを実行することで自動的に最適化されます：

```bash
# 1. アイキャッチ画像の自動設定
python scripts/active/add_featured_images.py

# 2. 記事改善の一括適用（メタディスクリプション、まとめ、内部リンク、本文画像）
python scripts/active/apply_post_improvements.py --mode all --dry-run
python scripts/active/apply_post_improvements.py --mode all
```

## 記事作成の流れ

1. **企画**: テーマ・キーワード・読者ターゲットを決定
2. **構成**: このテンプレートに沿ってアウトライン作成
3. **執筆**: WordPressエディタで記事本文を作成
4. **画像**: アイキャッチ画像を設定（Unsplash推奨）
5. **公開**: 記事を公開
6. **自動化**: スクリプトで自動最適化
7. **確認**: 表示確認・リンク確認
