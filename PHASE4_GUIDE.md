## ステップ1: AdSenseアカウント作成

### 1-1. AdSense公式サイトにアクセス

1. https://www.google.com/adsense にアクセス
2. 「ご利用開始」をクリック
3. Googleアカウントでログイン（Google Analyticsと同じアカウント推奨）

### 1-2. 基本情報の入力

#### 必要な情報
- **ウェブサイトのURL**: `https://techsimpleapp.main.jp`
- **メールアドレス**: AdSense関連の情報を受け取るメールアドレス
- **国または地域**: 日本
- **利用規約への同意**: チェックを入れる

#### 支払い情報（後で入力可能）
- **アカウントの種類**: 個人
- **氏名**: 正式な氏名（銀行口座名義と同じ）
- **住所**: 正確な住所（後で郵送される確認コードを受け取るため）
- **電話番号**: 連絡可能な電話番号

**注意**: 支払い情報は審査合格後でも入力可能ですが、早めに入力しておくとスムーズです。

## ステップ2: サイト登録

### 2-1. サイト情報の確認

AdSenseダッシュボードで以下を確認：
- サイトURL: `https://techsimpleapp.main.jp` が正しく登録されているか
- サイトの状態: 「審査準備中」または「審査待ち」

### 2-2. サイトのコンテンツ言語設定

1. AdSenseダッシュボード → **サイト** → **サイトを管理**
2. サイトの言語: **日本語** に設定
3. 保存

## ステップ3: 審査コード設置

審査コードは、あなたがサイトの所有者であることを証明するためのものです。

### 方法1: WordPressプラグイン使用（推奨・簡単）

#### Site Kit by Google プラグイン

1. **WordPressダッシュボード** → **プラグイン** → **新規追加**
2. 「Site Kit by Google」を検索してインストール・有効化
3. **Site Kit** → **設定**
4. Googleアカウントでログイン
5. **AdSense** を接続
6. 自動的に審査コードが設置されます ✅

**メリット**:
- コードを手動で貼り付ける必要なし
- Google Analytics、Search Consoleとも連携可能
- 審査後の広告管理も簡単

### 方法2: 手動でコード貼り付け（上級者向け）

#### 手順

1. AdSenseダッシュボードで審査コードをコピー
   ```html
   <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
        crossorigin="anonymous"></script>
   ```

2. **WordPressダッシュボード** → **外観** → **テーマエディター**

3. **警告**: テーマファイルを編集する前に必ずバックアップを取ってください

4. **header.php** を開く

5. `</head>` タグの直前に審査コードを貼り付け
   ```html
   <!-- Google AdSense 審査コード -->
   <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXXXXXX"
        crossorigin="anonymous"></script>
   </head>
   ```

6. **ファイルを更新** をクリック

### 方法3: Insert Headers and Footers プラグイン（中級者向け）

1. **WordPressダッシュボード** → **プラグイン** → **新規追加**
2. 「Insert Headers and Footers」を検索してインストール・有効化
3. **設定** → **Insert Headers and Footers**
4. **Scripts in Header** に審査コードを貼り付け
5. **Save** をクリック

### 3-1. 審査コード設置の確認

#### ブラウザでソースコード確認
1. サイトにアクセス: https://techsimpleapp.main.jp/
2. ページ上で右クリック → **ページのソースを表示**
3. `Ctrl+F`（または `Cmd+F`）で `adsbygoogle` を検索
4. 審査コードが見つかれば成功 ✅

#### AdSenseダッシュボードで確認
1. AdSenseダッシュボードに戻る
2. 「コードが見つかりました」と表示されるまで待つ（最大24時間）

## ステップ5: 審査結果待ち

### 審査期間
- **通常**: 1-2週間
- **場合によっては**: 数日〜1ヶ月

### 審査中にやること

#### 1. 通常運営を継続
- 新しい記事を投稿（週1-2記事推奨）
- 既存記事の更新・改善
- ユーザーからのコメントに返信

#### 2. やってはいけないこと
- サイトの大幅な変更（デザイン、構成）
- 記事の大量削除
- プライバシーポリシーの削除
- 審査コードの削除
- 不正クリック対策（自分でサイトを過剰にクリック）

#### 3. AdSenseダッシュボードを確認
- 週1-2回程度チェック
- 追加情報の要求がないか確認

### 審査結果の通知

#### 合格の場合 ✅
- メールで通知が届きます
- AdSenseダッシュボードに「承認されました」と表示
- 広告コードを設置できるようになります

#### 不合格の場合 ❌
- メールで不合格理由が通知されます
- よくある理由：
  - コンテンツ不足
  - 禁止コンテンツ
  - ナビゲーション不備
  - ポリシー違反

**再申請可能**: 不合格理由を改善後、2週間程度空けて再申請できます

## 🎉 審査合格後の手順

### 1. 広告コード設置

#### 自動広告（推奨）
1. AdSenseダッシュボード → **広告** → **サマリー**
2. **自動広告** を有効化
3. サイトに自動的に広告が表示されます

#### 手動広告（詳細設定）
1. AdSenseダッシュボード → **広告** → **広告ユニットごと**
2. 広告ユニット作成（ディスプレイ広告、記事内広告など）
3. 広告コードをコピー
4. WordPressの記事やウィジェットに貼り付け

### 2. 広告の最適化

#### 推奨配置
- **記事上**: タイトル直下
- **記事中**: H2見出しの前後
- **記事下**: まとめセクションの後
- **サイドバー**: サイドバーウィジェット

#### 注意事項
- 1ページあたりの広告数: 3-5個推奨（多すぎるとペナルティ）
- 誤クリック防止: ボタンの近くに広告を配置しない
- モバイル対応: モバイルでも見やすい配置

### 3. 収益の確認

#### AdSenseダッシュボード
- **ホーム**: 本日の収益、推定収益
- **レポート**: 詳細な収益レポート
- **最適化**: 広告の改善提案

#### 支払い
- **最低支払額**: 8,000円
- **支払い方法**: 銀行振込
- **支払い日**: 毎月21日頃（前月分）

## ⚠️ AdSense審査で注意すべきポイント

### 1. 禁止コンテンツ

以下のコンテンツは審査に通りません：
- **アダルトコンテンツ**: ポルノ、性的コンテンツ
- **暴力的コンテンツ**: グロテスク、残酷な内容
- **違法コンテンツ**: 違法ダウンロード、海賊版
- **誹謗中傷**: 特定の個人・団体への攻撃
- **アルコール・タバコ**: 販売促進
- **ギャンブル**: オンラインカジノ、競馬予想
- **著作権侵害**: 無断転載、コピペ

### 2. コンテンツの質

- **オリジナリティ**: 独自の視点、経験、分析
- **価値提供**: ユーザーに役立つ情報
- **正確性**: 誤情報、デマの排除
- **読みやすさ**: 適切な改行、見出し、画像

### 3. ユーザー体験

- **ナビゲーション**: 分かりやすいメニュー
- **サイト速度**: ページ速度50点以上
- **モバイル対応**: レスポンシブデザイン
- **広告配置**: ユーザー体験を損なわない

### 4. ポリシー遵守

- **プライバシーポリシー**: AdSense利用明記
- **Cookie同意**: EU圏からのアクセスがある場合
- **お問い合わせ**: 運営者への連絡手段

## 🔧 トラブルシューティング

### Q1: 審査コードが検出されない

**原因**:
- キャッシュプラグインの影響
- テーマの設定ミス
- コードの貼り付け位置が間違っている

**解決方法**:
1. キャッシュをクリア（ブラウザ＋WordPressプラグイン）
2. シークレットモードでサイトを開いてソースコード確認
3. Site Kit by Google プラグインを使用（推奨）

### Q2: 審査が長引く（2週間以上）

**原因**:
- サイトのコンテンツ量が少ない
- 更新頻度が低い
- 審査が混雑している

**解決方法**:
1. 新しい記事を追加（週1-2記事）
2. 既存記事を更新・改善
3. AdSenseフォーラムで質問

### Q3: 不合格通知を受けた

**原因**:
- コンテンツ不足
- ポリシー違反
- ナビゲーション不備

**解決方法**:
1. 不合格理由を詳しく確認
2. 該当箇所を改善
3. 2週間後に再申請

### Q4: 広告が表示されない（合格後）

**原因**:
- 広告コードの設置ミス
- キャッシュの影響
- AdSense側の審査中（広告配信まで最大24時間）

**解決方法**:
1. 広告コードが正しく設置されているか確認
2. キャッシュをクリア
3. 24時間待つ
