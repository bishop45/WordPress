**手順**（WordPress 管理画面で手動作業）:

#### ステップ1: 各記事の内容をコピー
1. WordPress 管理画面にログイン
2. `投稿` → `下書き` を開く
3. 各プライバシーポリシー記事を開き、**本文全体をコピー**

#### ステップ2: 固定ページを作成
1. `固定ページ` → `新規追加`
2. タイトル: 「つみたて習慣 - プライバシーポリシー」（アプリ名を含める）
3. コピーした本文を貼り付け
4. **公開**をクリック

#### ステップ3: noindex 設定
**Yoast SEO または All in One SEO Pack を使用している場合:**
1. 固定ページ編集画面の右側または下部にある SEO設定セクションを探す
2. 「検索エンジンがこのページをインデックスしないようにする」または「No Index」にチェック
3. 「No Follow」にもチェック（推奨）
4. **更新**をクリック

**SEO プラグインがない場合:**
- 固定ページのHTMLヘッダーに以下を追加:
```html
<meta name="robots" content="noindex, nofollow">
```

#### ステップ4: 元の記事を削除
1. `投稿` → `下書き` を開く
2. プライバシーポリシー系記事を選択
3. `ゴミ箱へ移動` または `完全に削除`

#### ステップ5: 確認
- 固定ページのURLをブラウザで開き、ページが正しく表示されることを確認
- ページのソースコードに `<meta name="robots" content="noindex` が含まれていることを確認

## 📞 困ったときの確認事項

### 記事が下書きに戻らない場合
- WordPress REST API の認証情報（.env ファイル）を確認
- WORDPRESS_APPLICATION_PASSWORD が正しいか確認
- WordPress の REST API が有効になっているか確認

### スクリプトがエラーになる場合
- 仮想環境を有効化しているか確認: `source venv/bin/activate`
- 依存パッケージがインストールされているか確認: `pip install -r requirements.txt`
- .env ファイルが存在し、環境変数が設定されているか確認

### AdSense 審査に再度不合格になった場合
- Google からのフィードバックを確認
- 不合格理由に基づいて、さらに記事の質を改善
- 2週間程度空けてから再申請
